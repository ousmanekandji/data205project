# data205project
This project analysis utilizes two primary datasets:
•	Food Inspection Data: Sourced from local government open data portals, this dataset includes attributes such as Business Name, Zip Code, Inspection Results, and specific violation markers ending in .C. (Critical Violations).
•	Closed Food Establishments: This dataset identifies businesses that were legally closed by health departments, providing names and closure dates.

Tools used:
R STUDIO
project structure:
•	Data Ingestion and Cleaning: Using R (tidyverse, stringr) for string normalization, handling missing values, and joining datasets.
•	Exploratory Data Analysis (EDA): Utilizing ggplot2 to create box plots and bar charts to identify trends.
